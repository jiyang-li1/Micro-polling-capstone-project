# Micro-polling-capstone-project

### 1 Setup

1 Setup the venv  
```{python}
python -m venv {venv name}
```
2 Activate the venv  
```{bash}
./{venv name}/Scripts/activate
```
3 install all the package in the venv  
 Backend framework and sql  
4 Run .py in venv  
5 Run the flask
```{bash}
flask
```
```{bash}
flask --app {name} run 
```

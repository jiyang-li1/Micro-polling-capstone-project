# Micro-polling-capstone-project

### 1 Setup

1 Setup the venv  
```{python}
python -m venv {venv name}
```
2 Activate the venv  
```{bash}
./{venv name}/Scripts/activate
```
3 install all the package in the venv(pip install)  
Flask and sqlalchemy
4 Run .py in venv  
Run db_init.py
model.py
main.py  
(Run all the other .py files to make sure youb have the test database)
